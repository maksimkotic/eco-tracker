# Полное описание структуры и функционала приложения Eco-Tracker

## Основные компоненты системы

### Ядро приложения
- `app.js` - Главный конфигурационный файл:
  - Инициализация Express.js
  - Настройка middleware
  - Подключение маршрутов
  - Обработка ошибок
- `.env` - Конфигурация окружения:
  - Параметры базы данных
  - Настройки сессий
  - Ключи безопасности
- `package.json` - Управление зависимостями:
  - Express.js
  - Sequelize (ORM)
  - EJS (шаблонизатор)
  - Пакеты аутентификации

### Модели данных (Sequelize)
- `User.js` - Пользователи:
  - Логин/пароль
  - Роли и права
  - Профильная информация
- `Habit.js` - Привычки:
  - Название и описание
  - Расписание выполнения
  - Статистика прогресса
- `Achievement.js` - Достижения:
  - Условия получения
  - Иконки и описания
  - Система бейджей
- `Checkin.js` - Отметки выполнения:
  - Дата и время
  - Статус выполнения
  - Связь с привычками
- `Role.js` - Система ролей:
  - Обычный пользователь
  - Модератор
  - Администратор

### Бизнес-логика
- `controllers/` - Обработчики запросов:
  - `authController.js` - Регистрация, вход, выход
  - `habitController.js` - CRUD для привычек
  - `achievementController.js` - Награды и прогресс
  - `profileController.js` - Управление профилем
  - `moderatorController.js` - Модерация контента
  - `adminController.js` - Администрирование системы

### Маршрутизация
- `routes/` - Система маршрутов Express:
  - Публичные роуты (авторизация)
  - Приватные роуты (личный кабинет)
  - Модераторские разделы
  - Административные панели

### Middleware
- `auth.js` - Проверка аутентификации
- `permissions.js` - Проверка прав доступа
- `role.js` - Управление ролями

## Интерфейсы и представления

### Шаблоны EJS
- Основные разделы:
  - `auth/` - Формы входа/регистрации
  - `habits/` - Трекер привычек
  - `profile/` - Личный кабинет
  - `admin/` - Панель администратора
  - `moderator/` - Панель модератора
  - `static/` - Информационные страницы

### Особенности интерфейса
- Адаптивный дизайн
- Система уведомлений
- Визуализация прогресса
- Интерактивные графики

## Вспомогательные системы

### Утилиты
- `utils/logger.js` - Система логирования:
  - Запись действий пользователей
  - Отслеживание ошибок
  - Ротация логов

### Логирование
- `logs/` - Хранение логов:
  - Access logs
  - Error logs
  - Audit logs

### Дополнительные модули
- Система достижений
- Календарь привычек
- Статистика и аналитика
- Система уведомлений

## Техническая реализация

### Стек технологий
- **Backend**:
  - Node.js + Express.js
  - Sequelize + SQLite
  - Сессионная аутентификация
- **Frontend**:
  - EJS шаблоны
  - Bootstrap 5
  - Chart.js для графиков
- **Инфраструктура**:
  - Система логирования
  - Менеджер ошибок
  - Резервное копирование

### Архитектурные особенности
- MVC паттерн
- Модульная структура
- Ролевая модель доступа
- Локализация ошибок
